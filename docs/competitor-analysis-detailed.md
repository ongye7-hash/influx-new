## 1. SNS서포터 (snssupporter.com)
**시장 포지션:** 최저가 전략, 박리다매

### 서비스 상세
- **인스타그램**: 팔로워, 좋아요, 상위노출, 자동 댓글
- **유튜브**: 조회수, 구독자, 댓글, 좋아요
- **페이스북**: 페이지 좋아요, 팔로워
- **틱톡**: 팔로워, 좋아요, 조회수
- **텔레그램**: 멤버, 조회수
- **카카오톡**: 플러스친구
- **쓰레드**: 팔로워, 좋아요
- **트위터**: 팔로워, 좋아요
- **N포털/플레이스**: 검색 최적화
- **이커머스**: 무신사, 쿠팡, 올리브영 리뷰/트래픽
- **특수 서비스**: 계정 육성 관리, 검색어 최적화, 블로그 배포

### 가격 정책
```
좋아요: 최소 0.5원/개
팔로워: 최소 5원/개
(도매가 수준 - 리셀러 타겟)
```

### 결제 시스템
- 잔액 예치 시스템 (선불 충전)
- 카드 결제 포인트 충전
- "잔액 부족 시 서비스 미제공" 정책

### 보증 정책
- **90일 이탈 보증**: 팔로워 이탈 시 무료 리필
- **3% 초과 제공**: 구매량 대비 추가 제공
- **24시간 자동 운영**: 상시 모니터링

### UI/UX 특징
- 6단계 주문 프로세스 (단순화)
- 플랫폼별 가이드 이미지 다수
- 실시간 주문 알림 티커
- 반응형 모바일/PC 레이아웃

### INFLUX 적용 포인트
```
✅ 90일 이탈 보증 → 신뢰도 핵심
✅ 3% 초과 제공 → 고객 만족도 상승
✅ 실시간 티커 → 활성화 느낌 연출
✅ 잔액 예치 시스템 → 선결제로 현금흐름 안정
```

- 명확한 CTA 버튼 ("주문하기")
- 반응형 그리드 레이아웃

### INFLUX 적용 포인트
