1단계: 소수에게 노출 (100-500명)
2단계: 반응 좋으면 확장 (1000-5000명)
3단계: 계속 좋으면 폭발 (10만+)

핵심은 1단계에서 좋은 반응을 얻는 것!

1️⃣ 완시율 (가장 중요!)
→ 영상을 끝까지 본 비율
→ 15초 영상 추천 (완시율 높음)

2️⃣ 반복 시청
→ 같은 영상 여러 번 봄
→ 루프되는 영상 만들기

3️⃣ 공유
→ 친구에게 보내고 싶은 콘텐츠

4️⃣ 댓글
→ 논쟁/공감 유발하는 주제

━━━━━━━━━━━━━━━━━━━━━━

📌 조회수 터지는 영상 공식

✅ 첫 1초 훅
"이거 모르면 손해예요"
"절대 하지 마세요"
"이렇게 하면 됩니다"

✅ 반전 구조
처음 예상과 다른 결말

✅ 정보 압축
1분 정보를 15초에 담기

✅ 트렌드 활용
