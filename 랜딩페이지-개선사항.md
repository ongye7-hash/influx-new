# 랜딩 페이지 전환율 개선 제안

## 현재 상태 분석
- 디자인: 깔끔하고 현대적
- 기능: 카운트다운, 시스템 로그, 성공 사례 등 구현됨
- 문제: 전환율 최적화 요소 부족

---

## 개선 사항

### 1. 히어로 섹션 CTA 강화
**현재:**
- "지금 시작하기" (일반적)

**개선안:**
- "2,000원 무료 크레딧 받고 시작하기 →"
- "30초 만에 무료 시작 (결제 정보 불필요)"

**효과:**
- 구체적 혜택 강조
- 진입 장벽 낮춤 (결제 정보 불필요)

---

### 2. 긴급성/희소성 요소 추가
**추가 요소:**
```
⏰ 오늘 가입 시 추가 혜택:
- 첫 주문 20% 할인
- 프리미엄 기능 7일 무료
- 24시간 이내 429명 남았습니다
```

**위치:** 히어로 섹션 하단

---

### 3. 신뢰 배지 추가
**추가 항목:**
- ✅ 30일 환불 보장
- ✅ SSL 보안 인증
- ✅ 99.8% 서비스 가동률
- ✅ 24/7 고객 지원

**위치:** CTA 버튼 바로 아래

**시각적:**
- 작은 아이콘 + 짧은 텍스트
- 한 줄로 배치

---

### 4. 실시간 주문 알림 팝업
**기능:**
- 하단 좌측에 작은 토스트 알림
- "김OO님이 방금 인스타 팔로워 1,000명 주문" (2-3초 표시)
- 3-5초 간격으로 랜덤 표시

**효과:**
- 사회적 증거 (다른 사람도 쓴다)
- 긴급성 강화 (지금 해야 함)

**구현:**
```tsx
const orderNotifications = [
  "김OO님이 방금 인스타 팔로워 1,000명 주문",
  "박OO님이 유튜브 구독자 5,000명 주문",
  "이OO님이 틱톡 조회수 50,000 주문",
  // ... 10개 정도
];
```

---

### 5. 가격 테이블 개선
**현재:** 가격 정보가 분산되어 있음

**개선안:**
- 명확한 가격표 (3-4개 인기 옵션)
- "가장 인기" 배지
- "초보자 추천" 배지
- 할인가 + 원가 표시 (~~49,000원~~ → **39,000원**)

**예시:**
```
┌─────────────────────────────────┐
│ 인스타 팔로워 1,000명            │
│ ~~39,000원~~ → 29,000원 (25% ↓)│
│ [가장 인기] [초보자 추천]       │
│ [지금 시작하기 →]               │
└─────────────────────────────────┘
```

---

### 6. FAQ 섹션 강화
**추가 질문:**
- ❓ 계정 정지 위험은 없나요?
  → 속도 제한 + 자연 패턴으로 안전합니다. 실제 정지 사례 0건

- ❓ 환불이 가능한가요?
  → 서비스 미진행 시 100% 환불 가능합니다.

- ❓ 얼마나 걸리나요?
  → 평균 24-72시간 내 완료됩니다.

---

### 7. Exit Intent 팝업
**조건:** 사용자가 페이지를 떠나려 할 때 (마우스가 상단으로 이동)

**내용:**
```
🎁 잠깐만요!
떠나기 전에 2,000원 무료 크레딧 받고 가세요

[이메일 입력]
[무료로 시작하기]

✅ 결제 정보 불필요
✅ 30초 가입
✅ 즉시 사용 가능
```

---

### 8. 모바일 최적화
**개선:**
- CTA 버튼 더 크게 (손가락 터치 영역)
- 스크롤 시 하단 고정 CTA 바
- 원탭 가입 (카카오/구글 로그인)

---

### 9. A/B 테스트 추천
**테스트 항목:**
- CTA 문구 ("지금 시작" vs "무료 크레딧 받기")
- 버튼 색상 (파란색 vs 초록색 vs 주황색)
- 가격 표시 (할인가 강조 vs 원가 제거)

---

### 10. 트러스트 시그널 추가
**섹션 추가:**
```
🔒 안전하고 투명한 서비스
- 서울 소재 법인 (사업자등록증 공개)
- 통신판매업 신고 완료
- 개인정보 보호 인증
- 24/7 고객 지원
```

**위치:** 푸터 위
