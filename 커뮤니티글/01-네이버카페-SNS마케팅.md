# SNS 마케팅 초보가 실수하는 5가지 (실전 경험 공유)

안녕하세요! 요즘 SNS 마케팅 본격적으로 시작하면서 시행착오 겪은 것들 공유합니다.

## 1. 팔로워/구독자만 늘리면 된다? ❌

처음엔 숫자만 늘리면 매출 나올 줄 알았어요.
→ 실제론 **참여율(좋아요, 댓글, 공유)**이 훨씬 중요

**해결책:**
- 인터랙션 유도 (질문, 투표)
- 스토리 적극 활용
- 댓글 답변 100%

## 2. 콘텐츠 없이 광고만 ❌

"우리 제품 사세요" 반복 → 언팔/차단
→ **가치 제공 80% + 광고 20%** 법칙

**실전 예시:**
- 유익한 정보 4개 → 제품 홍보 1개
- 스토리텔링으로 자연스럽게 녹이기

## 3. 모든 플랫폼 다 하기 ❌

인스타, 유튜브, 틱톡, 블로그 다 하려다가 다 중도포기...
→ **1-2개 플랫폼 집중** 후 확장

**추천 조합:**
- 인스타 + 블로그
- 유튜브 + 쇼츠
- 틱톡 + 인스타 릴스

## 4. 분석 안 하기 ❌

감으로 콘텐츠 올리기 → 성과 제자리
→ **인사이트 매주 체크**

**체크 항목:**
- 도달(Reach)
- 참여율(Engagement)
- 프로필 클릭
- 저장/공유

## 5. 초기 성장 너무 느림 ❌

자연 성장만 기다리면 6개월-1년 걸려요.
→ **초기 부스트로 시간 절약**

저는 INFLUX로 초기 팔로워 1000명 확보 → 알고리즘 부스트 받음
(자연 성장보다 3-4배 빠름)

**효과:**
- 신규 방문자들이 "많은 사람이 팔로우하네?" → 구독 확률 UP
- 알고리즘이 "성장세 있는 계정" 인식 → 추천 증가
